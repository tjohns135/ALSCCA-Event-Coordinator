# ALSCCA Class & PAX Reference
## Based on 2026 SCCA National Solo Rules + ALSCCA Event Data

---

## S1 (Street 1)
- SS — Super Street
- AS — A Street
- BS — B Street
- FS — F Street

## S2 (Street 2)
- CS — C Street
- ES — E Street

## S3 (Street 3)
- DS — D Street
- GS — G Street
- HS — H Street
- SSC — Solo Spec Coupe
- HCS — (unknown full name, keep as-is)

## ST (Street Touring)
- SST — Super Street Touring
- AST — A Street Touring
- BST — B Street Touring
- CST — C Street Touring
- DST — D Street Touring
- EST — E Street Touring
- GST — G Street Touring

## CAM (Classic American Muscle)
- CAMS — CAM Sport
- CAMC — CAM Contemporary
- CAMT — CAM Traditional

## XS (Xtreme Street)
- XA — Xtreme Street A
- XB — Xtreme Street B

## X (Pro)
- Self-designated class for the fast/experienced drivers
- Open — Any PAX allowed
- Appears as "Pro" in run group headers

## L (Ladies)
- Open — Any PAX allowed
- Not present at every event
- **Follow mode** (default): grouped with their PAX class's run group
- **Separate mode**: treated as their own class for run group assignment

## N (Novice)
- Open — Any PAX allowed
- **Follow mode** (default): grouped with their PAX class's run group (e.g., N_CST runs with ST)
- **Separate mode**: treated as their own class for run group assignment

## R (Race Tire)
- Primarily FSAE cars
- Catch-all for rare/unusual vehicles
- Open — Any PAX allowed

---

## Unmapped SCCA PAX Classes (rarely seen at ALSCCA events)

These exist in the 2026 SCCA National Solo Rules but rarely appear at local events. If an entrant registers with one of these, they would need to be manually assigned to a run group.

### Street Prepared Category
- SSP, CSP, DSP, ESP, FSP

### Street Modified Category
- SSM, SM, SMF

### Prepared Category
- XP, CP, DP, EP, FP

### Modified Category
- AM, BM, CM, DM, EM, FM
